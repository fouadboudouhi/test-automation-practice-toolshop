# Test Automation Practice Toolshop

This project demonstrates a realistic UI + API test automation setup
against https://practicesoftwaretesting.com.

## Scope
- UI Smoke & Regression tests using Robot Framework + Browser Library
- API tests using pytest
- Unified reporting via Allure

## What is NOT covered
- No backend data manipulation
- No payment or checkout validation
- No UI-based load testing

## Execution
See CI workflows or run locally via pytest / robot.
